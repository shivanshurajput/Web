Projects

Simon Game

The 'Simon Game' repository is a project that showcases my skills in web development using HTML, CSS, and JavaScript. It is a digital version of the classic Simon memory game, where players are challenged to remember and repeat a sequence of colored buttons. The game features an intuitive user interface, interactive gameplay, and sound effects to enhance the gaming experience. Through this project, I have demonstrated my ability to create engaging and interactive web applications using front-end technologies.

Drum Kit

The 'Drum Kit' repository is an exciting project that highlights my proficiency in web development using HTML, CSS, and JavaScript. It is a virtual drum set that allows users to play different drum sounds by pressing corresponding keys or clicking on the drum icons. The application offers an intuitive user interface with visually appealing drum images and responsive sound feedback. By showcasing my skills in front-end development, I have created an enjoyable and interactive experience for music enthusiasts.

My Blog

'MyBlog' is a personal portfolio and blogging project developed using Python Flask. The project incorporates HTML, CSS, and Bootstrap to create an attractive and user-friendly interface for visitors.
